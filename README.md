# AI Model Licenses

A reference list of AI model licenses and commercial use permissions.

## Data

[AIModelLicenses.csv](AIModelLicenses.csv) — AI models across image, video, text, and multimodal categories.

### Columns

| Column | Description |
|---|---|
| `Model Name` | Model name |
| `Model Type` | Image, Video, Text / LLM, Multimodal, 3D |
| `License Class` | `APACHE` · `MIT` · `RAIL` · `REVENUE_GATED` · `NON_COMMERCIAL` · `ENTERPRISE_REQUIRED` · `PROPRIETARY` |
| `License Description` | Human-readable license name |
| `Commercial Use` | `Allowed` · `Restricted` · `Not Allowed` · `API Only` · `Allowed <$1M` · `Allowed <$10M` |
| `License URL` | Link to the full license text |
